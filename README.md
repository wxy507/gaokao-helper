# 🎯 高考志愿填报助手

基于高考分数和省份的智能志愿填报推荐系统。提供完整的中国本科专业目录、院校排名、专业解析、就业前景和薪资数据。

## 功能特色

- **智能推荐**：选择省份和分数，自动推荐最适合的专业和院校
- **完整本科专业目录**：12大学科门类、800+本科专业，包含专业解析、就业去向、薪资情况
- **各大高校录取分数**：专业详情页展示开设该专业的院校及往年录取分数（从高到低排序）
- **全国院校大全**：100+高校信息排名，包含校园网站、公众号链接深入了解
- **毕业生去向**：院校详情页包含就业率、深造情况、主要雇主、薪资水平
- **志愿填报建议**：根据分数段和个人情况提供针对性策略建议
- **高考热点趋势**：结合当前中国国情和风口行业推荐专业方向

## 技术栈

- 纯静态 HTML/CSS/JavaScript
- GitHub Pages 部署
- 中国大陆无障碍访问（无 GFW 屏蔽资源）
- System fonts，无需加载外部字体

## 部署到 GitHub Pages

1. 创建 GitHub 仓库
2. 将本目录所有文件推送到仓库
3. 在仓库 Settings → Pages 中，选择 main 分支和 / (root) 目录
4. 等待部署完成后访问 `https://<用户名>.github.io/<仓库名>/`

也可直接访问 `index.html` 在本地预览。

## 数据说明

- 录取分数为往年参考数据，实际分数线每年会有变化
- 薪资数据为行业平均水平，因城市、企业、个人能力而异
- 院校排名综合参考多个权威排名
- 官方填报请以各省招办公布的招生计划和录取数据为准
